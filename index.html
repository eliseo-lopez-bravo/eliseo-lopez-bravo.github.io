IaC Terraform Docker nginx Demo

Description: This project demonstrates how to use Terraform to provision and manage Docker containers. In this example, I use Terraform to pull the latest Nginx image from Docker Hub and run a container on port 8000.

Key Files:

  • main.tf (Contains the Terraform configuration)

    • Commands:

  • terraform init (Initialize the Terraform project)
  • terraform plan (Plan and preview of changes to the Terraform project)
  • terraform apply (Apply the configuration and start the Docker container)
  • terraform destroy (Destroy the resources created by Terraform)

    • Outcome: Successfully running Nginx on http://localhost:8000 using Infrastructure as Code principles.

    Screenshots:

  • Terraform Apply
  • nginx up and running

    • GitHub Repo: iac-terraform-docker_nginx
